# FarmerPro

FarmerPro is an all-in-one application built for farmers. It helps you check balances, monitor account statuses, and automatically consolidate tokens into a single address.

Everything is done through a user-friendly interface — no coding required.

## What FarmerPro Can Do

- **EVM Wallet Checker** — check balances on Ethereum, Arbitrum, BNB Chain, and 90+ other networks
- **Solana Wallet Checker** — check SOL and token balances on Solana
- **Proxy Checker** — quickly find out which proxies are working and how well
- **Twitter Account Checker** — see whether an account is live or banned
- **Discord Account Checker** — see whether a token is active or not
- **Collector** — automatically gathers all tokens from all wallets and sends them to a target address
