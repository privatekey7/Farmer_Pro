# Modules

FarmerPro consists of six independent modules. Each module is a separate tab in the UI with its own settings panel and results table.

| Module | Purpose | Input |
|--------|---------|-------|
| [EVM Balance Checker](evm-balance-checker.md) | EVM wallet balances (DeBank API) | Addresses / keys |
| [SVM Balance Checker](svm-balance-checker.md) | Solana wallet balances | Addresses, Solana RPC |
| [Proxy Checker](proxy-checker.md) | Proxy quality check | Proxy list |
| [Twitter Checker](twitter-checker.md) | X.com token checker | Auth tokens, proxies |
| [Discord Checker](discord-checker.md) | Discord token checker | Tokens, proxies |
| [Collector](collector.md) | Token collection and migration | Private keys, subaccounts |
