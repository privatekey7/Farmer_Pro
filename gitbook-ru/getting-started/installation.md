# Установка

## Что понадобится

- Компьютер с Windows, macOS или Linux
- Python 3.11 или новее — скачай на [python.org](https://www.python.org/downloads/)

## Шаги установки

**1. Скачай FarmerPro**

Скачай архив с программой и распакуй его в любую папку.

**2. Открой терминал в папке с программой**

На Windows: зажми Shift и кликни правой кнопкой мыши в папке → "Открыть окно PowerShell здесь".

**3. Создай виртуальное окружение**

Это изолированная копия Python специально для FarmerPro. Она не мешает другим программам.

```
python -m venv .venv
```

Затем активируй его:

- Windows: `.venv\Scripts\activate`
- macOS / Linux: `source .venv/bin/activate`

После активации в начале строки появится `(.venv)`.

**4. Установи зависимости**

```
pip install -r requirements.txt
```

Подожди пока всё скачается — это займёт 1–3 минуты.

**5. Запусти программу**

```
python -m app.main
```

Откроется окно FarmerPro.

{% hint style="warning" %}
На Windows может появиться сообщение о нехватке Microsoft Visual C++. Скачай его бесплатно с сайта Microsoft и повтори установку.
{% endhint %}

{% hint style="info" %}
Каждый раз при запуске нужно сначала активировать окружение (шаг 3), а потом запускать программу (шаг 5).
{% endhint %}
