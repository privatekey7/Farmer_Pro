# Конфигурация

## Переменные окружения (.env)

| Переменная | Описание | По умолчанию |
|-----------|---------|-------------|
| `LOG_LEVEL` | Уровень логирования: `DEBUG`, `INFO`, `WARNING`, `ERROR` | `INFO` |

Пример `.env`:

```env
LOG_LEVEL=INFO
```

## Параметры модулей

Каждый модуль настраивается через панель в UI — отдельный конфиг-виджет слева от таблицы результатов. Параметры сохраняются на время сессии.

### Общие параметры (все модули)

| Параметр | Описание |
|---------|---------|
| Wallets / Items | Путь к текстовому файлу или вставка вручную |
| Proxies | Путь к файлу прокси (обязательно) |
| Concurrency | Количество параллельных задач (asyncio Semaphore) |

### Collector — дополнительные параметры

| Параметр | Описание | По умолчанию |
|---------|---------|-------------|
| `min_token_usd` | Минимальная стоимость токена для свапа ($) | 0.03 |
| `min_bridge_usd` | Минимальная сумма для бриджа ($) | 0.03 |
| `excluded_chains` | Сети, которые нужно пропустить | [] |
| `target_chains` | Целевые сети (DeBank-ключи или названия) | [] |
| `slippage` | Допустимое проскальзывание (%) | 3.0 |
| `delay_min` | Минимальная пауза между кошельками (сек) | 60 |
| `delay_max` | Максимальная пауза между кошельками (сек) | 180 |
| `send_to_exchange` | Отправлять на субаккаунт биржи | false |
| `delay_after_bridge` | Пауза перед отправкой на биржу (сек) | 60 |

## Логи

Логи пишутся в `logs/farmerpro.log` и отображаются в нижней панели UI в реальном времени. Приватные ключи и токены автоматически маскируются.
