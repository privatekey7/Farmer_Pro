# Что нужно подготовить

Перед работой тебе понадобится несколько файлов. Все они — обычные текстовые файлы (.txt), по одной записи на строку.

## Файл с кошельками

Это список адресов или ключей кошельков, которые ты хочешь проверить.

FarmerPro понимает три варианта:

**Только адрес** — подходит для проверки балансов:
```
0xAbCd1234...
0x9876Ef12...
```

**Приватный ключ** — нужен для Collector (чтобы отправлять транзакции):
```
0x4f3ca1b2...
0xd8e72f90...
```

**Мнемоника** (seed-фраза из 12 или 24 слов) — тоже подходит для Collector:
```
abandon abandon abandon abandon abandon abandon abandon abandon abandon abandon abandon legal
```

{% hint style="info" %}
Для проверки балансов достаточно обычных адресов. Для Collector нужны приватные ключи или мнемоники — программа должна уметь подписывать транзакции от имени кошелька.
{% endhint %}

## Файл с прокси

Прокси — это промежуточные серверы, через которые программа делает запросы. Они нужны чтобы сервисы не блокировали тебя при массовой проверке.

{% hint style="danger" %}
Прокси обязательны для всех модулей. Без них сервисы заблокируют твой IP уже после первых нескольких запросов. Купить прокси можно здесь: [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488).
{% endhint %}

Примеры форматов прокси:

```
# Без логина и пароля
192.168.1.1:8080

# С логином и паролем
192.168.1.1:8080:myuser:mypass

# Альтернативный формат
myuser:mypass@192.168.1.1:8080

# С указанием протокола
http://myuser:mypass@192.168.1.1:8080
```

{% hint style="warning" %}
Используй только HTTP-прокси. SOCKS5-прокси не поддерживаются модулем Proxy Checker. Для Twitter и Discord нужны резидентные прокси — они выглядят как обычные домашние подключения и реже блокируются.
{% endhint %}

## Файл с токенами Twitter или Discord

Токен (или auth_token) — это специальный код, который твой браузер использует чтобы доказать что ты вошёл в аккаунт. Его можно найти в куках браузера.

Как найти auth_token Twitter:
1. Войди в Twitter в браузере
2. Открой инструменты разработчика (F12)
3. Перейди в раздел Application → Cookies
4. Найди cookie с именем `auth_token` — скопируй его значение

Файл выглядит так — один токен на строку:
```
abc123def456ghi789...
xyz987wvu654...
```

Для Discord токен находится там же, или его можно получить из настроек приложения.

## Файл субаккаунтов биржи (только для Collector)

Это список EVM-адресов, на которые Collector будет отправлять собранные средства. Обычно это депозитные адреса на бирже — отдельные для каждого кошелька.

```
0xBirzha111...
0xBirzha222...
0xBirzha333...
```

Collector сопоставляет их по порядку: первый кошелёк → первый адрес биржи, второй кошелёк → второй адрес и так далее.

{% hint style="info" %}
Строки начинающиеся с `#` игнорируются во всех файлах. Используй это для комментариев или чтобы временно исключить запись.
{% endhint %}
