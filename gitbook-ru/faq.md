# Частые вопросы

## Программа не запускается — что делать?

Скорее всего Python не установлен или не добавлен в PATH. Скачай Python 3.11+ с [python.org](https://www.python.org/downloads/) и при установке поставь галочку **"Add Python to PATH"**. Затем повтори `pip install -r requirements.txt`.

На Windows иногда нужен дополнительный пакет — Microsoft Visual C++ Redistributable. Скачай его бесплатно с сайта Microsoft.

---

## Почему все результаты показывают ERROR?

Почти всегда это проблема с прокси:
- Прокси не загружены — убедись что файл выбран
- Прокси не работают — сначала проверь их в Proxy Checker
- Прокси из дата-центров — для Twitter и Discord нужны резидентные [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488)

---

## Можно ли запускать без прокси?

Нет. Без [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488) сервисы заблокируют твой IP уже после первых нескольких запросов. Прокси обязательны для всех модулей.

---

## Сколько прокси нужно?

Зависит от объёма работы. Желательно 1к1. FarmerPro автоматически распределяет нагрузку между ними по очереди.

---

## Все прокси показывают `low` в Proxy Checker — это нормально?

Если у тебя прокси из дата-центра (AWS, Hetzner, DigitalOcean) — да, они почти всегда `low`. Для высокого качества нужны резидентные [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488) — это прокси с реальных домашних IP-адресов.

---

## Как остановить модуль во время работы?

Нажми кнопку **Стоп**. Модуль закончит текущую задачу и остановится. Уже полученные результаты сохранятся в таблице.

---

## Приложение зависло — что делать?

Не закрывай окно принудительно — это может прервать транзакции в Collector. Дождись завершения текущей задачи, затем нажми Стоп.

Если программа реально зависла и не отвечает — закрой через диспетчер задач. Результаты незавершённой сессии могут потеряться.

---

## Collector работает очень долго — это нормально?

Да. Collector обрабатывает кошельки по одному и делает паузы между ними (по умолчанию 1–3 минуты). Это сделано намеренно — реальные транзакции требуют времени, и паузы снижают риск блокировок. Для 100 кошельков это может занять несколько часов.

---

## FarmerPro хранит мои ключи или токены?

Нет. FarmerPro не отправляет твои данные никуда. Приватные ключи и токены используются только локально на твоём компьютере для выполнения запросов. В логах они автоматически скрываются (`***`).
