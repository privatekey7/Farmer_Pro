# Collector

Автоматически собирает все активы с кошельков и переводит их в одно место — например, на биржевой депозит.

**Что происходит для каждого кошелька:**
1. Все не-нативные токены (USDC, USDT и т.д.) обмениваются на нативный токен сети (ETH, BNB и т.д.)
2. Если на кошельке есть токены но не хватает нативки, он сам её забриджит из доступной сети чтобы произвести сбор.
3. Нативный токен переводится в нужную тебе сеть (например, всё в Arbitrum)
4. Опционально — переводится на указанный адрес биржи

## Что нужно

- Файл с **приватными ключами или мнемониками** (только адреса не подходят — нужна возможность подписывать транзакции)
- Резидентные [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488) (обязательно)
- Файл с адресами биржи — если хочешь отправлять на субаккаунты (опционально)

## Настройки

| Параметр | Что делает | По умолчанию |
|---------|------------|-------------|
| **Мин. стоимость токена** | Токены дешевле этой суммы не трогаются | $0.03 |
| **Мин. сумма для перевода** | Если нативного токена меньше — перевод пропускается | $0.03 |
| **Исключить сети** | Выбранные сети будут пропущены | — |
| **Целевые сети** | Куда переводить (выбери одну или несколько) | — |
| **Slippage** | Допустимое отклонение цены при обмене (%) | 3% |
| **Пауза между кошельками** | Случайная задержка, чтобы не выглядеть как бот | 60–180 сек |
| **Отправить на биржу** | Включает шаг перевода на субаккаунт | выкл |
| **Пауза перед отправкой** | Ожидание после перевода в сеть | 60 сек |

## Как запустить

1. Открой вкладку **Collector**
2. Загрузи файл с приватными ключами
3. Загрузи файл с прокси
4. Выбери **целевые сети** — куда собирать токены
5. При необходимости настрой остальные параметры
6. Если нужна отправка на биржу — включи галочку и загрузи файл с адресами
7. Нажми **Старт**

{% hint style="warning" %}
Кошельки обрабатываются по одному, не параллельно. Это сделано специально — чтобы транзакции не конфликтовали между собой. Для большого количества кошельков это займёт время.
{% endhint %}

## Что ты получишь

| Колонка | Что означает |
|---------|-------------|
| `address` | Адрес кошелька |
| `chains_processed` | Сети, где были найдены токены |
| `chains_skipped` | Пропущенные сети и причина |
| `tokens_swapped` | Сколько обменов выполнено |
| `total_collected_usd` | Суммарная стоимость в USD |
| `bridge_chain` | Откуда и куда сделан перевод |
| `bridge_status` | Результат перевода |
| `exchange_tx` | Хэш транзакции на биржу |

## После завершения

Нажми **CSV**, **JSON** или **XLSX** для сохранения результатов. Кнопки экспорта появляются автоматически после окончания работы.
