# SVM Balance Checker

Проверяет балансы кошельков на Solana — SOL и все SPL-токены.

## Что нужно

- Файл с Solana-адресами (формат base58, начинаются не с `0x`)
- Файл с [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488) (обязательно)
- Список RPC-серверов Solana (адреса серверов, через которые идут запросы)

{% hint style="warning" %}
Бесплатный публичный RPC Solana (`api.mainnet-beta.solana.com`) может быть не стабильным, если не чекает пробуй другой RPC.
{% endhint %}

## Как запустить

1. Открой вкладку **SVM Balance Checker**
2. Загрузи файл с кошельками
3. Загрузи файл с прокси
4. В поле **Solana RPC** вставь адрес своего RPC-сервера
5. Нажми **Старт**

## Что ты получишь

| Колонка | Что означает |
|---------|-------------|
| `address` | Адрес кошелька Solana |
| `sol_balance` | Сколько SOL на кошельке |
| `sol_usd` | Стоимость SOL в долларах |
| `tokens` | Количество SPL-токенов |
| `top_tokens` | Самые дорогие токены |
