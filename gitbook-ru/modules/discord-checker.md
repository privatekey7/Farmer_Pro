# Discord Checker

Проверяет Discord-токены — действительны ли они и какая информация привязана к аккаунту.

## Что нужно

- Файл с токенами (один токен на строку)
- Резидентные [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488) (обязательно)

{% hint style="danger" %}
Прокси обязательны. Discord блокирует массовые запросы с одного IP. Используй резидентные [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488).
{% endhint %}

## Как запустить

1. Открой вкладку **Discord Checker**
2. Загрузи файл с токенами
3. Загрузи файл с прокси
4. Нажми **Старт**

## Что ты получишь

| Колонка | Что означает |
|---------|-------------|
| `token` | Начало токена (остальное скрыто) |
| `username` | Имя пользователя |
| `user_id` | Уникальный ID аккаунта |
| `email` | Email (если привязан) |
| `has_phone` | Привязан ли номер телефона |
| `status` | Статус токена |

### Статусы

| Статус | Что это значит |
|--------|---------------|
| `OK` | Токен живой, аккаунт активен |
| `INVALID` | Токен недействителен |
| `DISABLED` | Аккаунт отключён Discord |
| `ERROR` | Ошибка запроса — обычно проблема с прокси |
