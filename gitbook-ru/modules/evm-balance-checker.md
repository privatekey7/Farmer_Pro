# EVM Balance Checker

Проверяет балансы кошельков на Ethereum, Arbitrum, BNB Chain, Optimism, Base и ещё 90+ совместимых сетях — всё за один запуск.

## Что нужно

- Файл с кошельками (адреса кошельков)
- Файл с [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488) (обязательно)

## Как запустить

1. Открой вкладку **EVM Balance Checker**
2. Нажми **Обзор** рядом с "Кошельки" и выбери свой файл
3. Нажми **Обзор** рядом с "Прокси" и выбери файл с прокси
4. Нажми **Старт**

Результаты появляются в таблице по мере проверки.

## Что ты получишь

| Колонка | Что означает |
|---------|-------------|
| `address` | Адрес кошелька |
| `total_usd` | Суммарная стоимость всего на кошельке в долларах |
| `tokens_count` | Сколько разных токенов найдено |
| `chains` | В каких сетях есть активы |
| `top_tokens` | Самые дорогие токены на кошельке |

## Как сохранить результат

После завершения нажми кнопку **CSV**, **JSON** или **XLSX** — выбери формат и место сохранения.

Два режима сохранения:
- **Summary** — одна строка на кошелёк (удобно для общей сводки)
- **Tokens** — отдельная строка на каждый токен (удобно для детального анализа)

{% hint style="info" %}
Хочешь экспортировать только конкретный токен? В поле "Токены" есть выпадающий список который покажет все токены которые есть на кошельках и можно экспортировать только нужный токен в конкретной сети.
{% endhint %}
