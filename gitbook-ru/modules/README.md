# Модули

FarmerPro состоит из шести независимых модулей. Каждый модуль — это отдельная вкладка в UI со своей панелью настроек и таблицей результатов.

| Модуль | Назначение | Входные данные |
|--------|-----------|---------------|
| [EVM Balance Checker](evm-balance-checker.md) | Балансы EVM-кошельков (DeBank API) | Адреса / ключи |
| [SVM Balance Checker](svm-balance-checker.md) | Балансы Solana-кошельков | Адреса, Solana RPC |
| [Proxy Checker](proxy-checker.md) | Проверка качества прокси | Список прокси |
| [Twitter Checker](twitter-checker.md) | Чекер токенов X.com | Auth-токены, прокси |
| [Discord Checker](discord-checker.md) | Чекер токенов Discord | Токены, прокси |
| [Collector](collector.md) | Сбор и миграция токенов | Приватные ключи, субаккаунты |
