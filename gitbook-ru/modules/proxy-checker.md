# Proxy Checker

Проверяет список прокси — работают ли они, насколько быстрые и насколько "чистые" (не видно ли что это прокси).

{% hint style="danger" %}
Proxy Checker работает только с **HTTP-прокси**. SOCKS5-прокси не поддерживаются.
{% endhint %}

## Что нужно

- Файл с HTTP-прокси

## Как запустить

1. Открой вкладку **Proxy Checker**
2. Загрузи файл с прокси
3. Нажми **Старт**

## Что ты получишь

| Колонка | Что означает |
|---------|-------------|
| `proxy` | Адрес прокси |
| `status` | `OK` — работает, `ERROR` — не работает |
| `quality` | Насколько "чистый" прокси |
| `latency_ms` | Скорость ответа в миллисекундах |

### Уровни качества

| Качество | Что это значит |
|---------|---------------|
| `high` | Отлично — прокси не определяется как прокси |
| `medium` | Нормально — прокси работает, но частично виден |
| `low` | Плохо — прокси явно из дата-центра, легко блокируется |

{% hint style="info" %}
Для Twitter и Discord нужны прокси с качеством `high`. Для проверки балансов EVM подойдёт любой рабочий прокси. Купить качественные [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488).
{% endhint %}
