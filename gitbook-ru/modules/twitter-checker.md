# Twitter Checker

Проверяет Twitter-токены — живой аккаунт или нет, и какой у него статус.

**Twitter-токен** (auth_token) — это код из твоего браузера, который подтверждает что ты вошёл в аккаунт. Как его найти:
1. Войди в Twitter в браузере
2. Нажми F12 → вкладка **Application** → **Cookies** → `twitter.com`
3. Найди строку `auth_token` и скопируй её значение

## Что нужно

- Файл с токенами (один токен на строку)
- Резидентные [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488) (обязательно)

{% hint style="danger" %}
Прокси обязательны. Без них Twitter заблокирует твой IP уже после первых запросов. Используй резидентные [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488) — они выглядят как обычные домашние подключения.
{% endhint %}

## Как запустить

1. Открой вкладку **Twitter Checker**
2. Загрузи файл с токенами
3. Загрузи файл с прокси
4. Нажми **Старт**

## Что ты получишь

| Колонка | Что означает |
|---------|-------------|
| `token` | Первые символы токена (остальное скрыто) |
| `username` | Имя аккаунта (@username) |
| `status` | Статус аккаунта |

### Статусы

| Статус | Что это значит |
|--------|---------------|
| `ok` | Аккаунт живой и активный |
| `invalid` | Токен недействителен или устарел |
| `suspended` | Аккаунт заблокирован Twitter навсегда |
| `locked` | Аккаунт временно заморожён — нужно пройти проверку |
| `error` | Ошибка запроса — обычно проблема с прокси |
