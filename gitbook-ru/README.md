# FarmerPro

FarmerPro — это приложение в котором есть всё что нужно для фармеров. Она помогает проверять балансы, следить за состоянием аккаунтов и автоматически собирать токены в одно место.

Всё делается через удобный интерфейс — никакого кода писать не нужно.

## Что умеет FarmerPro

- **Проверка EVM-кошельков** — узнай балансы на Ethereum, Arbitrum, BNB Chain и ещё 90+ сетях
- **Проверка Solana-кошельков** — балансы SOL и токенов на Solana
- **Проверка прокси** — быстро узнай, какие прокси работают и насколько хорошо
- **Проверка Twitter-аккаунтов** — живой аккаунт или заблокирован
- **Проверка Discord-аккаунтов** — активен токен или нет
- **Collector** — автоматически собирает все токены со всех кошельков и отправляет на нужный адрес
