# Устранение неполадок

## Collector

### Модуль не запускается: "Нет поддерживаемых сетей"

**Причина:** LI.FI `/chains` и Relay `/chains` оба вернули ошибку при инициализации.

**Решение:**
1. Проверь интернет-соединение
2. Убедись, что прокси (если используются) работоспособны — запусти Proxy Checker
3. Попробуй запустить позже — временная недоступность внешних API

---

### "Кошельков X, субаккаунтов Y"

**Причина:** Включена галочка `send_to_exchange`, но количество строк в файлах не совпадает.

**Решение:** Убедись, что в `wallets.txt` и `subaccounts.txt` одинаковое количество строк. Collector назначает субаккаунты по индексу (1-й кошелёк → 1-й субаккаунт).

---

### Bridge завис на статусе PENDING

Collector опрашивает статус до **30 минут**. Если за это время статус не перешёл в `DONE` — транзакция помечается как `FAILED`, шаг отправки на биржу пропускается.

Для проверки транзакции вручную используй хэш из колонки `bridge_tx`.

---

### Ошибка "balance too low" в логах

Нативного токена на кошельке недостаточно для покрытия газа или значение после вычета газа стало ≤ 0. Модуль пропускает шаг бриджа для этого кошелька.

---

## EVM Balance Checker

### Мало результатов / пустые балансы

**Причина:** DeBank API может временно не возвращать данные при высокой нагрузке.

**Решение:**
- Снизь `Concurrency` до 5–10
- Используй прокси для распределения запросов
- При ошибках включается retry (до 10 попыток, пауза 3 сек)

---

## SVM Balance Checker

### "Rate limit exceeded" на Solana RPC

Публичный RPC `api.mainnet-beta.solana.com` строго ограничен.

**Решение:** Укажи приватный RPC (Helius, QuickNode, Triton) в поле "Solana RPC URLs". Один RPC на строку.

---

## Proxy Checker

### Все прокси показывают `low`

Прокси из дата-центров (AWS, DO, Hetzner) почти всегда определяются как `low`. Для высокого качества (`high`) требуются **резидентные** [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488).

---

## Twitter / Discord Checker

### Массовые ошибки `error` / `INVALID`

Запросы блокируются по IP — прокси не указаны или используются прокси из дата-центров.

**Решение:** Прокси обязательны. Используй резидентные [прокси](https://dashboard.travchisproxies.com/billing/aff.php?aff=1488) — дата-центры блокируются Twitter и Discord особенно агрессивно.

## Общие проблемы

### UI не реагирует во время работы модуля

Это нормально — обработка выполняется в фоновом потоке. UI обновляется по мере поступления результатов. Для остановки нажми **Стоп** — модуль завершит текущую итерацию и выйдет.

---

### Приложение не запускается на Windows

Убедись, что установлены:
- Python 3.11+ (из официального установщика, не Microsoft Store)
- Microsoft Visual C++ Redistributable (x64)
- Все зависимости: `pip install -r requirements.txt`

{% hint style="info" %}
При ошибке `No module named 'PySide6'` — выполни `pip install PySide6` в активированном виртуальном окружении.
{% endhint %}
