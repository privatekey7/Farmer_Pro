# Форматы файлов

Все файлы — обычные текстовые (.txt), одна запись на строку. Строки начинающиеся с `#` игнорируются.

## Кошельки

```
# Адреса
0xAbCd1234567890abcd1234567890AbCd12345678

# Приватные ключи
0x4f3ca1b2d8e72f904f3ca1b2d8e72f904f3ca1b2d8e72f904f3ca1b2d8e72f90

# Мнемоника (12 или 24 слова)
abandon abandon abandon abandon abandon abandon abandon abandon abandon abandon abandon legal
```

## Прокси

```
# Только адрес:порт
192.168.1.1:8080

# Адрес:порт:логин:пароль
192.168.1.1:8080:myuser:mypass

# логин:пароль@адрес:порт
myuser:mypass@192.168.1.1:8080

# С явным протоколом
http://myuser:mypass@192.168.1.1:8080
socks5://myuser:mypass@192.168.1.1:1080
```

{% hint style="warning" %}
Для Proxy Checker используй только HTTP-прокси. SOCKS5 там не работает.
{% endhint %}

## Субаккаунты биржи (для Collector)

```
# EVM-адреса получателей, по одному на строку
0x1111111111111111111111111111111111111111
0x2222222222222222222222222222222222222222
0x3333333333333333333333333333333333333333
```

{% hint style="info" %}
Порядок важен: первый кошелёк → первый адрес биржи, второй → второй, и т.д. Количество строк должно совпадать.
{% endhint %}
