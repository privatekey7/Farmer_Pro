# DexScreener

Используется для получения цен токенов Solana в модуле **SVM Balance Checker**.

## Назначение

| Модуль | Роль |
|--------|------|
| SVM Balance Checker | Получает USD-цены SPL-токенов и SOL для расчёта `sol_usd` |

## Как работает

`SolanaClient` запрашивает цены батчами по 30 mint-адресов за раз:

```
GET https://api.dexscreener.com/tokens/v1/solana/<mint1>,<mint2>,...
```

Включает адрес нативного SOL (`So111...112`) для получения текущего курса SOL/USD.

## Особенности

- Запросы выполняются пакетами (30 mint-адресов за запрос)
- Для известных токенов используется локальный кэш `app/data/solana_tokens.json`
- Если цена для токена не найдена — устанавливается `0.0` (токен включается в список без USD-стоимости)

{% hint style="info" %}
DexScreener не требует API-ключа для базовых ценовых запросов.
{% endhint %}
