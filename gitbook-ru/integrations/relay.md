# Relay

Альтернативный bridge-провайдер. Используется в Collector как запасной вариант или основной (настраивается через `bridge_provider`).

## Используемые эндпоинты

| Эндпоинт | Назначение |
|---------|-----------|
| `GET /chains` | Список поддерживаемых сетей |
| `POST /quote` | Получение квоты на bridge |
| `GET /status` | Статус транзакции |

## Отличия от LI.FI

- Polling статуса: каждые **5 секунд** (LI.FI — 15 сек)
- Квоты через POST, а не GET
- Поддержка `gas_on_destination` — газ в целевой сети включается в bridge

## Адрес платы

Адрес комиссии Relay хранится в обфусцированном виде (XOR-шифрование). Это защита от случайного раскрытия в логах.

## Получение квоты

```python
quote = await relay_client.get_quote(
    from_chain=1,
    to_chain=42161,
    from_amount="500000000000000000",  # 0.5 ETH в wei
    from_address="0x...",
    to_address="0x..."
)
```

{% hint style="info" %}
Relay специализируется на bridging нативных токенов (ETH → ETH в другой сети). Для swap токенов перед bridge используй LI.FI.
{% endhint %}
